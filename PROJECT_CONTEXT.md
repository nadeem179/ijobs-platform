# Trusted Hiring Platform

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

## Product Vision
Modern trusted hiring platform inspired by:
- LinkedIn Jobs
- Indeed
- Upwork

Focus:
- verified hiring
- trust-first UX
- modern minimal UI
- clean product flow
- no spam experience

## UX Rules
- mobile-first
- low clutter
- premium spacing
- calm typography
- no fake marketing sections
- no giant SaaS hero blocks
- no corporate HR dashboard feel

## Current Product Structure
Routes:
- /
- /jobs
- /jobs/[id]
- /recruiter
- /profile

## Current Focus
Phase 6:
- homepage refinement
- navigation flow
- trust communication
- CTA routing
- spacing improvements

## Next Focus
Phase 7:
- mock auth state

- profile dropdown
- recruiter protection

## Important Product Rules
- Users should explore before signup
- Auth should be lightweight
- Job cards should be highly scannable
- Homepage should preview actual product
- Trust indicators are core UX

## Avoid
- giant empty sections
- fake startup metrics
- overdesigned gradients
- excessive animations
- cluttered layouts
- dead buttons
